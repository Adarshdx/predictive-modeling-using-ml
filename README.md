# Predictive Modeling Using Machine Learning

## Overview

This project demonstrates the implementation of a Machine Learning predictive model using Python and Scikit-learn. The objective is to predict student performance based on academic scores and evaluate the model's effectiveness.

## Features

- Data preprocessing and preparation
- Train-test dataset splitting
- Linear Regression model training
- Prediction of outcomes
- Model evaluation using MAE and R² Score
- Visualization of actual vs predicted values

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Project Workflow

1. Load and prepare dataset
2. Split data into training and testing sets
3. Train Linear Regression model
4. Generate predictions
5. Evaluate model performance
6. Visualize results

## Evaluation Metrics

- Mean Absolute Error (MAE)
- R² Score
